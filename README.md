# Meu Primeiro Projeto Completo com React + Tailwind CSS

O foco foi praticar React do jeito certo, com código limpo e componentizado. Implementei:

- Componentes reutilizáveis e organizados
- Gerenciamento de estado com useState
- Troca dinâmica da imagem principal do iPhone conforme o botão de cor é clicado
- Animações suaves na transição das imagens com estilização completa com Tailwind CSS
- Renderização dinâmica usando o método .map() em arrays de objetos

Foi um ótimo exercício para reforçar a componentização, manipulação de estado de forma previsível e o poder do Tailwind pra estilizar rápido e com consistência.
